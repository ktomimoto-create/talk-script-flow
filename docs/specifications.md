# 仕様書：トークスクリプトフロー

## プロジェクト概要
株式会社フルタイムシステムの電話対応を補助するための、インタラクティブなトークスクリプトフローアプリケーション。

## 技術スタック
- **Frontend**: React (スタンドアロン版は CDN 経由、開発版は Vite)
- **Language**: TypeScript / JavaScript (Babel)
- **Styling**: Vanilla CSS (Glassmorphism デザイン)
- **Database / Backend**: Firebase (Cloud Firestore)
- **SSO Authentication**: Microsoft Entra ID (Azure AD)
- **Development Port**: 5177 (Vite 開発サーバー)

## アプリケーション構成

1.  **受電 (Main Entry)**: 最初のエントリーポイント。「受電」とその下の8カテゴリー。
2.  **フロー図 (地図)**: 各カテゴリーごとの主な転送先・用件を表示。管理会社、管理員、協力会社、建築、連動相手など、特定カテゴリに応じた分岐を視覚化。
3.  **詳細ヒアリング階層**: 各項目をクリックした際に表示される具体的な確認事項。
4.  **最終アクション**: 不要になったため削除（元の転送完了画面）。
5.  **メモ入力スペース**: 管理会社、管理員、協力会社、建築、連動相手のヒアリングページにおいて、オペレーターが情報を一時的に記録できるフォームを表示。
    - **カテゴリによる項目名の自動切り替え**:
        - 建築 / 連動相手: 「現場名」「会社名/氏名」「内容」「折り返し先」
        - 管理会社 / 管理員 / 協力会社: 「号機/物件名」「会社名/氏名」「内容」「折り返し先」

### 特徴
- **インタラクティブ・マップ**: 全体像を構造的に表示し、オペレーターが視覚的に判断・操作できるインターフェース。
- **下部メモバー（架電メモ＆個人メモの一体化表示）**: 画面下部にスッキリした1行のコンパクトなバーを常時表示。
    - **中央 (架電メモ)**: 電話番号、架電先氏名、現場/物件名を入力して「不通」履歴として保存できる機能。入力エリア全体を画面の中央にセンタリング配置し、過去の不通履歴一覧（パネル）もその直上の画面中央にポップアップ表示されるように設計。
    - **右側 (個人メモ表示)**: 「個人メモ履歴」ボタンを右端に配置。
        - ボタンを押すと、下部バーの上に「個人メモ履歴一覧パネル」がふわっと浮き出て表示される（不通履歴と同様のポップアップパネル）。
        - パネル内には、ログインユーザーが登録した個人メモが簡易リスト（1行）形式で表示される（初期画面では全件、用件ボックス選択中は該当ボックスに自動で絞り込み）。
        - パネル内の各リスト項目をクリックすると、画面中央にグラスモーフィズムデザインの「個人メモ詳細」ポップアップモーダルが開き、詳細情報（現場名、会社名、用件内容、折り返し番号、保存日時）の確認、およびその場での削除・閉じる操作が可能。
    - **自動照合**: 受電メモの「折り返し先」入力時に保存済み架電データと自動照合して架電者をバッジで表示。
- **ブラウザ履歴連携**: ブラウザの「戻る」「進む」ボタンに対応。各状態がURLハッシュ（#id）として保存。詳細ウィンドウが開いている状態での別の用件のクリックは履歴を上書き（replaceState）し、サイドバー内の「戻る」ボタンを押した際は一発で初期画面（受電スタート）に戻ってウィンドウを閉じる設計に最適化。
- **動的カラーリング**: 選択したカテゴリーに応じたテーマカラーが適用。
- **洗練されたグループ表示**: 中間カテゴリー（ディスパッチャー、施工管理等）は、背景の囲みや枠線を廃止し、テキストのみのフラットなデザインで表示。
- **動的コネクタ**: ボックスの数や配置に合わせ、接続線を動的に生成。
- **SSO 認証 (Microsoft Entra ID)**: Microsoft アカウントによるサインイン機能を搭載。
    - **AuthGate**: 未ログイン時はサインイン画面を表示。
    - **プロフィールの表示**: 右上チップにユーザー名とメールアドレスを表示。
- **データ保護と自動削除**:
    - **プライバシー**: 架電メモはサインインしたユーザー本人のもののみが表示される。
    - **24時間自動削除**: 保存から24時間が経過したメモは一覧から自動的に除外される。
- **操作性向上 (Hover Effect)**: フロー図内の各サブ項目（詳細な用件）にカーソルを合わせた際、背景の強調、拡大、シャドウ付与が行われ、選択中であることが明確に分かるように改善。
- **画面サイズの固定化**: スケールを「中（PC）」の比率に固定し、常に安定したレイアウトで表示されるように設計。
